Writing Controllers

Learn how to create controllers that handle HTTP requests and orchestrate application flow

What is a Controller?

A controller is a class responsible for handling HTTP requests, processing input, calling business logic, and returning responses. Controllers act as the bridge between the HTTP layer and your application's business logic.

Controller's Role in the Framework

Controller Layer (YOU ARE HERE)
  • Receives HTTP requests from the Router
  • Validates and sanitizes input data
  • Calls Service methods to execute business logic
  • Formats and returns JSON responses
  • Handles exceptions and errors
Service Layer
  • Contains business rules and logic
  • Orchestrates multiple repository calls
  • Handles transactions
Repository Layer
  • Executes database queries
  • Maps data to arrays/objects

What Controllers Should NOT Do

  • ❌ Contain business logic (use Services)
  • ❌ Write SQL queries directly (use Repositories)
  • ❌ Perform complex data transformations
  • ❌ Access the database directly (except for simple cases)

BaseController Class

All controllers in this framework must extend BaseController, which provides essential functionality and helper methods.

What BaseController Provides

  • Database Access: $this->db - Pre-configured Database instance
  • Environment Variables: Automatically loaded from .env
  • Security: Automatic HTTPS enforcement (except for CLI)
  • Response Helpers: Pre-built methods for JSON responses
  • Validation Helpers: Built-in request validation
  • Input Sanitization: Automatic XSS protection

BaseController Constructor (Automatic)

// Happens automatically when you extend BaseController

public function __construct() {
    // 1. Load environment variables from .env
    $this->ensureEnvironmentLoaded();
    
    // 2. Enforce HTTPS (except CLI scripts)
    if (php_sapi_name() !== 'cli') {
        Security::ensureSecure();
    }
    
    // 3. Initialize database connection
    try {
        $pdo = DatabaseConnection::pdo();
        $this->db = new Database($pdo);
    } catch (\Exception $e) {
        error_log("Database initialization failed: " . $e->getMessage());
        $this->db = null;
    }
}
You get all of this for free! Just extend BaseController and you have database access, environment variables, and security automatically configured.

Creating Your First Controller

File Location and Naming

  • Location: src/api/controllers/
  • Naming: *Controller.php (e.g., ProductController.php)
  • Class Name: Must match filename (e.g., class ProductController)

Basic Controller Template

<?php
// src/api/controllers/ProductController.php

namespace App\Controllers;

use App\Core\BaseController;
use App\Core\Exceptions\ValidationException;
use App\Core\Exceptions\NotFoundException;
use App\Services\ProductService;

class ProductController extends BaseController
{
    private ProductService $productService;
    
    public function __construct()
    {
        parent::__construct();  // IMPORTANT: Call parent constructor
        
        // Initialize your services
        $this->productService = new ProductService($this->db);
    }
    
    // Your controller methods go here
    
    public function list()
    {
        // Handle GET /products/list
    }
    
    public function details($id)
    {
        // Handle GET /products/{id}
    }
    
    public function create()
    {
        // Handle POST /products/create
    }
}
Important: Always call parent::__construct() in your controller's constructor to ensure the database and environment are initialized.

Getting Request Data

1. URL Parameters (from Route)

// Route definition:
router->add([
    'method' => 'GET',
    'url' => '/products/{id}',
    'controller' => 'ProductController@details'
]);

// Controller method:
public function details($id)
{
    // $id is automatically passed as a method parameter
    // URL: /api/products/123
    // $id = "123"
    
    try {
        $product = $this->productService->getProduct($id);
        $this->sendSuccess('Product retrieved', $product);
    } catch (NotFoundException $e) {
        $this->sendError($e->getMessage(), 404);
    }
}

2. Query String Parameters (GET)

// URL: /api/products/list?category=electronics&limit=20

public function list()
{
    // Get all query parameters as an associative array
    $params = $this->getQueryParams();
    
    // $params = [
    //     'category' => 'electronics',
    //     'limit' => '20'
    // ]
    
    // Automatically sanitized for XSS protection
    
    $products = $this->productService->getProducts($params);
    $this->sendSuccess('Products retrieved', $products);
}

3. Request Body (POST/JSON)

// POST /api/products/create
// Body: {"name": "Laptop", "price": 50000, "stock": 10}

public function create()
{
    // Get request body data (works with JSON or form data)
    $data = $this->getRequestData();
    
    // $data = [
    //     'name' => 'Laptop',
    //     'price' => 50000,
    //     'stock' => 10
    // ]
    
    // Automatically sanitized for XSS protection
    
    try {
        $product = $this->productService->createProduct($data);
        $this->sendSuccess('Product created', $product);
    } catch (ValidationException $e) {
        $this->sendValidationError($e->getMessage(), $e->getErrors());
    }
}

Request Data Methods Reference

Method Purpose Returns
getRequestData() Get POST/JSON body data Sanitized array
getQueryParams() Get URL query parameters (?key=value) Sanitized array
sanitizeInput($input) Manually sanitize data (XSS protection) Sanitized string/array

Sending Responses

BaseController provides four main response methods that return properly formatted JSON responses.

sendSuccess() - Success Responses (200 OK)

protected function sendSuccess(string $message, $data = null, array $extra = []): void

Use for: Successful operations

// Simple success
$this->sendSuccess('Product deleted successfully');

// Success with data
$this->sendSuccess('Product retrieved', $product);

// Success with extra metadata
$this->sendSuccess('Products retrieved', $products, [
    'total' => 150,
    'page' => 1,
    'perPage' => 10
]);
Response Format:
{
    "status": "success",
    "message": "Products retrieved",
    "data": [...],
    "total": 150,
    "page": 1,
    "perPage": 10
}

sendError() - Error Responses (400, 404, etc.)

protected function sendError(string $message, int $statusCode = 400, $data = null): void

Use for: Client errors, not found, forbidden, etc.

// Generic error (400)
$this->sendError('Invalid product ID');

// Not found (404)
$this->sendError('Product not found', 404);

// Forbidden (403)
$this->sendError('You do not have permission to delete this product', 403);

// Error with additional data
$this->sendError('Stock insufficient', 400, [
    'available' => 5,
    'requested' => 10
]);
Response Format:
{
    "status": "error",
    "message": "Product not found"
}

// HTTP Status: 404

sendValidationError() - Validation Errors (422)

protected function sendValidationError(string $message, array $errors): void

Use for: Form validation failures

$this->sendValidationError('Validation failed', [
    'name' => 'Product name is required',
    'price' => 'Price must be greater than 0',
    'sku' => 'SKU already exists'
]);
Response Format:
{
    "status": "error",
    "message": "Validation failed",
    "validation_errors": {
        "name": "Product name is required",
        "price": "Price must be greater than 0",
        "sku": "SKU already exists"
    }
}

// HTTP Status: 422

sendServerError() - Server Errors (500)

protected function sendServerError(string $message = 'Internal server error'): void

Use for: Unexpected server-side errors

try {
    $product = $this->productService->createProduct($data);
    $this->sendSuccess('Product created', $product);
} catch (\Exception $e) {
    error_log('Product creation failed: ' . $e->getMessage());
    $this->sendServerError('Failed to create product');
}
Response Format:
{
    "status": "error",
    "message": "Failed to create product"
}

// HTTP Status: 500

Validation in Controllers

Method 1: validateRequired() - Simple Validation

public function create()
{
    $data = $this->getRequestData();
    
    // Check for required fields
    $required = ['name', 'price', 'sku'];
    $missing = $this->validateRequired($data, $required);
    
    if (!empty($missing)) {
        $errors = [];
        foreach ($missing as $field) {
            $errors[$field] = "The {$field} field is required";
        }
        $this->sendValidationError('Missing required fields', $errors);
        return;
    }
    
    // Continue with creation...
    $product = $this->productService->createProduct($data);
    $this->sendSuccess('Product created', $product);
}

Method 2: validateRequest() - Advanced Validation

public function create()
{
    $data = $this->getRequestData();
    
    // Define validation rules
    $rules = [
        'name' => 'required|string|min:3|max:255',
        'email' => 'required|email',
        'price' => 'required|numeric|min:0',
        'stock' => 'required|integer|min:0',
        'category' => 'string'
    ];
    
    // Validate using ValidationUtility
    $errors = $this->validateRequest($rules);
    
    if (!empty($errors)) {
        $this->sendValidationError('Validation failed', $errors);
        return;
    }
    
    // Data is valid, proceed
    $product = $this->productService->createProduct($data);
    $this->sendSuccess('Product created', $product);
}

Available Validation Rules

Rule Description Example
required Field must be present and not empty 'name' => 'required'
email Must be a valid email address 'email' => 'required|email'
numeric Must be a number 'price' => 'numeric'
integer Must be an integer 'stock' => 'integer'
string Must be a string 'name' => 'string'
min:n Minimum value/length 'price' => 'min:0'
max:n Maximum value/length 'name' => 'max:255'

Exception Handling

The framework provides custom exception classes for different error scenarios.

Available Exception Classes

Exception Status Code When to Use
NotFoundException 404 Resource not found
ValidationException 422 Validation errors
DatabaseException 500 Database errors

Exception Handling Pattern

use App\Core\Exceptions\NotFoundException;
use App\Core\Exceptions\ValidationException;
use App\Core\Exceptions\DatabaseException;

public function details($id)
{
    try {
        // Call service method
        $product = $this->productService->getProduct($id);
        $this->sendSuccess('Product retrieved', $product);
        
    } catch (NotFoundException $e) {
        // Handle not found (404)
        $this->sendError($e->getMessage(), 404);
        
    } catch (ValidationException $e) {
        // Handle validation errors (422)
        $this->sendValidationError($e->getMessage(), $e->getErrors());
        
    } catch (DatabaseException $e) {
        // Handle database errors (500)
        error_log('Database error: ' . $e->getMessage());
        $this->sendServerError('Database error occurred');
        
    } catch (\Exception $e) {
        // Handle unexpected errors (500)
        error_log('Unexpected error: ' . $e->getMessage());
        $this->sendServerError('An unexpected error occurred');
    }
}

Complete Controller Examples

ProductController - Full Implementation

<?php

namespace App\Controllers;

use App\Core\BaseController;
use App\Core\Exceptions\ValidationException;
use App\Core\Exceptions\NotFoundException;
use App\Services\ProductService;

class ProductController extends BaseController
{
    private ProductService $productService;
    
    public function __construct()
    {
        parent::__construct();
        $this->productService = new ProductService($this->db);
    }
    
    /**
     * GET /products/list
     * Get list of products with optional filters
     */
    public function list()
    {
        try {
            // Get query parameters
            $filters = $this->getQueryParams();
            
            // Call service
            $products = $this->productService->getProducts($filters);
            
            // Send success response
            $this->sendSuccess('Products retrieved', $products);
            
        } catch (\Exception $e) {
            error_log('Error fetching products: ' . $e->getMessage());
            $this->sendServerError('Failed to retrieve products');
        }
    }
    
    /**
     * GET /products/{id}
     * Get single product by ID
     */
    public function details($id)
    {
        try {
            // Call service
            $product = $this->productService->getProduct($id);
            
            // Send success response
            $this->sendSuccess('Product retrieved', $product);
            
        } catch (NotFoundException $e) {
            $this->sendError($e->getMessage(), 404);
            
        } catch (\Exception $e) {
            error_log('Error fetching product: ' . $e->getMessage());
            $this->sendServerError('Failed to retrieve product');
        }
    }
    
    /**
     * POST /products/create
     * Create a new product
     */
    public function create()
    {
        try {
            // Get request data
            $data = $this->getRequestData();
            
            // Validate required fields
            $required = ['name', 'sku', 'price'];
            $missing = $this->validateRequired($data, $required);
            
            if (!empty($missing)) {
                $errors = [];
                foreach ($missing as $field) {
                    $errors[$field] = "The {$field} field is required";
                }
                $this->sendValidationError('Missing required fields', $errors);
                return;
            }
            
            // Call service to create product
            $product = $this->productService->createProduct($data);
            
            // Send success response
            $this->sendSuccess('Product created successfully', $product);
            
        } catch (ValidationException $e) {
            $this->sendValidationError($e->getMessage(), $e->getErrors());
            
        } catch (\Exception $e) {
            error_log('Error creating product: ' . $e->getMessage());
            $this->sendServerError('Failed to create product');
        }
    }
    
    /**
     * POST /products/update
     * Update an existing product
     */
    public function update()
    {
        try {
            // Get request data
            $data = $this->getRequestData();
            
            // Validate product ID
            if (empty($data['productid'])) {
                $this->sendValidationError('Product ID is required', [
                    'productid' => 'Product ID is required'
                ]);
                return;
            }
            
            // Call service to update product
            $product = $this->productService->updateProduct($data['productid'], $data);
            
            // Send success response
            $this->sendSuccess('Product updated successfully', $product);
            
        } catch (NotFoundException $e) {
            $this->sendError($e->getMessage(), 404);
            
        } catch (ValidationException $e) {
            $this->sendValidationError($e->getMessage(), $e->getErrors());
            
        } catch (\Exception $e) {
            error_log('Error updating product: ' . $e->getMessage());
            $this->sendServerError('Failed to update product');
        }
    }
}

OrderController - Full Implementation

<?php

namespace App\Controllers;

use App\Core\BaseController;
use App\Core\Exceptions\ValidationException;
use App\Core\Exceptions\NotFoundException;
use App\Services\OrderService;

class OrderController extends BaseController
{
    private OrderService $orderService;
    
    public function __construct()
    {
        parent::__construct();
        $this->orderService = new OrderService($this->db);
    }
    
    /**
     * POST /orders/create
     * Create a new order with items
     */
    public function create()
    {
        try {
            // Get request data
            $data = $this->getRequestData();
            
            // Validate required fields
            $required = ['userid', 'paymentmethod', 'items'];
            $missing = $this->validateRequired($data, $required);
            
            if (!empty($missing)) {
                $errors = [];
                foreach ($missing as $field) {
                    $errors[$field] = "The {$field} field is required";
                }
                $this->sendValidationError('Missing required fields', $errors);
                return;
            }
            
            // Validate items array
            if (empty($data['items']) || !is_array($data['items'])) {
                $this->sendValidationError('Invalid order data', [
                    'items' => 'Items must be a non-empty array'
                ]);
                return;
            }
            
            // Extract order data and items
            $orderData = [
                'userid' => $data['userid'],
                'paymentmethod' => $data['paymentmethod'],
                'totalamount' => $data['totalamount'] ?? 0,
                'notes' => $data['notes'] ?? null
            ];
            
            $items = $data['items'];
            
            // Call service to create order
            $order = $this->orderService->createOrder($orderData, $items);
            
            // Send success response
            $this->sendSuccess('Order created successfully', $order);
            
        } catch (ValidationException $e) {
            $this->sendValidationError($e->getMessage(), $e->getErrors());
            
        } catch (\Exception $e) {
            error_log('Error creating order: ' . $e->getMessage());
            $this->sendServerError('Failed to create order');
        }
    }
    
    /**
     * GET /orders/details
     * Get order details by ID
     */
    public function details()
    {
        try {
            // Get order ID from query params
            $params = $this->getQueryParams();
            $orderId = $params['orderid'] ?? null;
            
            if (empty($orderId)) {
                $this->sendValidationError('Order ID is required', [
                    'orderid' => 'Order ID is required'
                ]);
                return;
            }
            
            // Call service
            $order = $this->orderService->getOrderDetails($orderId);
            
            // Send success response
            $this->sendSuccess('Order details retrieved', $order);
            
        } catch (NotFoundException $e) {
            $this->sendError($e->getMessage(), 404);
            
        } catch (\Exception $e) {
            error_log('Error fetching order: ' . $e->getMessage());
            $this->sendServerError('Failed to retrieve order details');
        }
    }
    
    /**
     * POST /orders/cancel
     * Cancel an order and restore stock
     */
    public function cancel()
    {
        try {
            // Get request data
            $data = $this->getRequestData();
            
            // Validate required fields
            $required = ['orderid', 'reason'];
            $missing = $this->validateRequired($data, $required);
            
            if (!empty($missing)) {
                $errors = [];
                foreach ($missing as $field) {
                    $errors[$field] = "The {$field} field is required";
                }
                $this->sendValidationError('Missing required fields', $errors);
                return;
            }
            
            // Call service to cancel order
            $this->orderService->cancelOrder($data['orderid'], $data['reason']);
            
            // Send success response
            $this->sendSuccess('Order cancelled successfully');
            
        } catch (NotFoundException $e) {
            $this->sendError($e->getMessage(), 404);
            
        } catch (ValidationException $e) {
            $this->sendValidationError($e->getMessage(), $e->getErrors());
            
        } catch (\Exception $e) {
            error_log('Error cancelling order: ' . $e->getMessage());
            $this->sendServerError('Failed to cancel order');
        }
    }
}

Controller Best Practices

✓ DO

Keep controllers thin - delegate logic to services
Always call parent::__construct()
Use try-catch blocks for exception handling
Validate input before calling services
Use appropriate response methods (sendSuccess, sendError, etc.)
Log errors with error_log()
Return early on validation failures
Use type hints for service dependencies

✗ DON'T

Write business logic in controllers
Access database directly (except simple cases)
Forget to handle exceptions
Return raw arrays or strings - use response methods
Mix HTML rendering with API controllers
Hardcode values - use environment variables
Skip input validation

Common Controller Patterns

Pattern 1: List with Pagination/Filters

public function list()
{
    try {
        $filters = $this->getQueryParams();
        
        // Extract pagination params
        $page = $filters['page'] ?? 1;
        $limit = $filters['limit'] ?? 10;
        
        // Get data from service
        $result = $this->productService->getProducts($filters);
        
        // Send with metadata
        $this->sendSuccess('Products retrieved', $result['data'], [
            'total' => $result['total'],
            'page' => $page,
            'limit' => $limit
        ]);
        
    } catch (\Exception $e) {
        $this->sendServerError('Failed to retrieve products');
    }
}

Pattern 2: Create with Validation

public function create()
{
    try {
        $data = $this->getRequestData();
        
        // Validate
        $required = ['name', 'price'];
        $missing = $this->validateRequired($data, $required);
        if (!empty($missing)) {
            $errors = array_fill_keys($missing, 'This field is required');
            $this->sendValidationError('Validation failed', $errors);
            return;
        }
        
        // Create
        $product = $this->productService->createProduct($data);
        $this->sendSuccess('Product created', $product);
        
    } catch (ValidationException $e) {
        $this->sendValidationError($e->getMessage(), $e->getErrors());
    } catch (\Exception $e) {
        $this->sendServerError('Failed to create product');
    }
}

Pattern 3: Update Existing Resource

public function update()
{
    try {
        $data = $this->getRequestData();
        
        // Validate ID
        if (empty($data['id'])) {
            $this->sendError('Product ID is required', 400);
            return;
        }
        
        // Update
        $product = $this->productService->updateProduct($data['id'], $data);
        $this->sendSuccess('Product updated', $product);
        
    } catch (NotFoundException $e) {
        $this->sendError($e->getMessage(), 404);
    } catch (ValidationException $e) {
        $this->sendValidationError($e->getMessage(), $e->getErrors());
    } catch (\Exception $e) {
        $this->sendServerError('Failed to update product');
    }
}

Pattern 4: Delete with Confirmation

public function delete()
{
    try {
        $data = $this->getRequestData();
        
        // Validate ID
        if (empty($data['id'])) {
            $this->sendError('Product ID is required', 400);
            return;
        }
        
        // Delete
        $this->productService->deleteProduct($data['id']);
        $this->sendSuccess('Product deleted successfully');
        
    } catch (NotFoundException $e) {
        $this->sendError($e->getMessage(), 404);
    } catch (\Exception $e) {
        $this->sendServerError('Failed to delete product');
    }
}
