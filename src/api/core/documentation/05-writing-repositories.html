Writing Repositories

Learn how to implement the data access layer with repositories

What is a Repository?

A Repository is a class that handles all database operations for a specific entity or domain. Repositories are the only place in your application where SQL queries should be written.

Three-Layer Architecture

🔵 Controller Layer
  • Handles HTTP requests and responses
  • Calls Service methods only
  • Returns formatted JSON responses
🟢 Service Layer
  • Contains all business logic
  • Orchestrates repositories
  • Manages transactions
🟡 Repository Layer — YOU ARE HERE
  • Executes SQL queries via the Database class
  • Maps database results to arrays
  • Provides data access methods
  • Absolutely no business logic

Why Repositories Exist

Benefits of Repository Pattern

  • Single Source of Truth: All SQL for an entity lives in one file
  • Testability: Easy to mock repositories in unit tests
  • Maintainability: Changing a query only affects one file
  • Security: Centralized parameter binding prevents SQL injection
  • Reusability: Same query methods can be reused across multiple services

Repository Layer Responsibilities

✓ Repositories SHOULD

Write SQL queries (SELECT, INSERT, UPDATE, DELETE)
Use parameter binding for ALL user input
Return arrays or null — never PDO objects
Provide clean CRUD methods (Create, Read, Update, Delete)
Map database results to meaningful array structures
Handle database-specific operations like JOINs and aggregates

✗ Repositories SHOULD NOT

Contain any business logic or validation rules
Handle transactions (services are responsible for that)
Call other repositories directly
Throw ValidationException — only DatabaseException if truly needed
Calculate derived values (e.g., totals, discounts)
Know anything about HTTP requests or responses

Creating Your First Repository

File Location and Naming

  • Location: src/api/repositories/
  • Filename: *Repository.php — e.g., ProductRepository.php
  • Class Name: Must exactly match the filename — e.g., class ProductRepository
  • Namespace: App\Repositories

Basic Repository Template

<?php
// src/api/repositories/ProductRepository.php

namespace App\Repositories;

use App\Core\Database;

class ProductRepository
{
    private Database $db;

    /**
     * Constructor - Database is injected automatically by the framework
     */
    public function __construct(Database $db)
    {
        $this->db = $db;
    }

    /**
     * Find product by ID
     */
    public function findById(int $id): ?array
    {
        return $this->db->first(
            "SELECT * FROM products WHERE id = :id",
            ['id' => $id]
        );
    }

    /**
     * Get all products
     */
    public function findAll(): array
    {
        return $this->db->select("SELECT * FROM products ORDER BY createdat DESC");
    }

    /**
     * Create new product — returns the new record's ID
     */
    public function create(array $data): int
    {
        return $this->db->insert(
            "INSERT INTO products (name, sku, price, stock, category)
             VALUES (:name, :sku, :price, :stock, :category)",
            [
                'name'     => $data['name'],
                'sku'      => $data['sku'],
                'price'    => $data['price'],
                'stock'    => $data['stock'] ?? 0,
                'category' => $data['category'] ?? null,
            ]
        );
    }

    /**
     * Update product — returns true if a row was changed
     */
    public function update(int $id, array $data): bool
    {
        $affected = $this->db->update(
            "UPDATE products SET name = :name, price = :price, stock = :stock WHERE id = :id",
            [
                'id'    => $id,
                'name'  => $data['name'],
                'price' => $data['price'],
                'stock' => $data['stock'],
            ]
        );
        return $affected > 0;
    }

    /**
     * Delete product — returns true if a row was deleted
     */
    public function delete(int $id): bool
    {
        return $this->db->delete(
            "DELETE FROM products WHERE id = :id",
            ['id' => $id]
        ) > 0;
    }
}

Using the Database Class

The Database class (src/api/core/Database.php) is a PDO wrapper that provides helper methods for all operations. Every repository receives a Database instance via constructor injection. All methods use exception-based error handling — if an operation fails it throws a DatabaseException, so you never need to check a 'success' flag.

select() — Fetch Multiple Rows

public function select(string $sql, array $params = []): array
Returns: array (rows) Empty [] when nothing found

Use for: SELECT queries that return zero or more rows. Identical to query() — both names work.

// All products (no params needed)
$products = $this->db->select("SELECT * FROM products ORDER BY createdat DESC");
// Returns: [['id' => 1, 'name' => 'Laptop'], ['id' => 2, 'name' => 'Phone']]

// Filter by category
$products = $this->db->select(
    "SELECT * FROM products WHERE category = :category AND status = 'active'",
    ['category' => 'Electronics']
);

// With ordering and limit
$products = $this->db->select(
    "SELECT * FROM products ORDER BY price ASC LIMIT :limit",
    ['limit' => 20]
);

// If nothing found, returns empty array (never null)
if (empty($products)) {
    // handle no results
}

first() — Fetch a Single Row

public function first(string $sql, array $params = []): ?array
Returns: array (row found) Returns: null (not found)

Use for: Looking up a single record by ID or unique key. Automatically appends LIMIT 1 if not present, so it's also more efficient than select().

// Find product by primary key
$product = $this->db->first(
    "SELECT * FROM products WHERE id = :id",
    ['id' => 42]
);

if ($product === null) {
    throw new NotFoundException('Product not found');
}

// Find by unique field (email)
$user = $this->db->first(
    "SELECT * FROM users WHERE email = :email",
    ['email' => $email]
);

// Safe to access directly after null check
echo $product['name'];
echo $product['price'];

insert() — Insert a Record

public function insert(string $sql, array $params = []): int
Returns: int (New Auto-Increment ID)

Use for: INSERT queries. Returns the new record's ID directly — no array unpacking needed. If the insert fails, a DatabaseException is thrown automatically.

// Simple insert — returns the new ID directly
$productId = $this->db->insert(
    "INSERT INTO products (name, sku, price, stock)
     VALUES (:name, :sku, :price, :stock)",
    [
        'name'  => 'Wireless Keyboard',
        'sku'   => 'WK-2024',
        'price' => 1500,
        'stock' => 50,
    ]
);
// $productId = 123 (the new row's auto-increment ID)

// Use the returned ID immediately
$newProduct = $this->db->first(
    "SELECT * FROM products WHERE id = :id",
    ['id' => $productId]
);

update() — Update Records

public function update(string $sql, array $params = []): int
Returns: int (Number of Affected Rows)

Use for: UPDATE queries. Returns the count of rows actually changed. A result of 0 means either the record was not found, or the values were already identical to what you tried to set.

$affected = $this->db->update(
    "UPDATE products SET price = :price, stock = :stock WHERE id = :id",
    ['id' => 123, 'price' => 1200, 'stock' => 45]
);

// $affected = 1  → row found and updated
// $affected = 0  → row not found OR values were already the same

// Convert to bool for repository return type
return $affected > 0;

// Conditional update (only update if stock is enough)
$sold = $this->db->update(
    "UPDATE products SET stock = stock - :qty WHERE id = :id AND stock >= :qty",
    ['id' => $productId, 'qty' => $quantity]
);
if ($sold === 0) {
    throw new \Exception('Insufficient stock');
}

delete() — Delete Records

public function delete(string $sql, array $params = []): int
Returns: int (Number of Affected Rows)

Use for: DELETE queries. Returns the count of rows removed. Returns 0 if the record did not exist.

$affected = $this->db->delete(
    "DELETE FROM products WHERE id = :id",
    ['id' => 123]
);

// $affected = 1  → product was found and deleted
// $affected = 0  → product didn't exist

return $affected > 0; // true/false for the repository caller

// Delete with conditions
$affected = $this->db->delete(
    "DELETE FROM sessions WHERE userid = :userid AND expiredat < NOW()",
    ['userid' => $userId]
);
// Returns how many expired sessions were removed

Parameter Binding (SQL Injection Prevention)

⚠️ CRITICAL: Always Use Parameter Binding

Parameter binding prevents SQL injection attacks. Never concatenate user input directly into SQL strings. PDO automatically escapes all bound values.

Named Parameters — Correct Way

// ✅ GOOD — Named parameters, completely safe
$sql = "SELECT * FROM products WHERE category = :category AND price > :minPrice";
$products = $this->db->select($sql, [
    'category' => $userInput,    // Safely escaped by PDO
    'minPrice' => $minPrice,     // Safely escaped by PDO
]);

// ❌ BAD — String concatenation, VULNERABLE TO SQL INJECTION
$sql = "SELECT * FROM products WHERE category = '$userInput'";
$products = $this->db->select($sql); // NEVER DO THIS!

How Parameter Binding Works

  1. Write your SQL using :placeholder syntax for every variable value
  2. Pass a matching associative array as the second argument
  3. PDO prepares the statement, then safely binds and escapes each value
  4. SQL injection is structurally impossible — user input can never break the query structure
// Full example: safe INSERT with user-supplied data
$sql = "INSERT INTO users (name, email, age) VALUES (:name, :email, :age)";

$params = [
    'name'  => $userData['name'],    // User input → safely escaped
    'email' => $userData['email'],   // User input → safely escaped
    'age'   => $userData['age'],     // User input → safely escaped
];

$userId = $this->db->insert($sql, $params);
// Even if $userData['name'] = "Robert'); DROP TABLE users; --"
// PDO treats it as a literal string value, not SQL

Complete Repository Examples

ProductRepository — Full Implementation

<?php
// src/api/repositories/ProductRepository.php

namespace App\Repositories;

use App\Core\Database;

/**
 * ProductRepository
 * All data access for the `products` table lives here.
 */
class ProductRepository
{
    private Database $db;

    public function __construct(Database $db)
    {
        $this->db = $db;
    }

    // ----------------------------------------------------------------
    // READ Methods
    // ----------------------------------------------------------------

    /**
     * Find a product by its primary key.
     *
     * @param int $id
     * @return array|null  Product row or null if not found
     */
    public function findById(int $id): ?array
    {
        return $this->db->first(
            "SELECT * FROM products WHERE id = :id",
            ['id' => $id]
        );
    }

    /**
     * Get all products, newest first.
     *
     * @param int $limit  Max rows to return (default 50)
     * @return array
     */
    public function findAll(int $limit = 50): array
    {
        return $this->db->select(
            "SELECT * FROM products ORDER BY createdat DESC LIMIT :limit",
            ['limit' => $limit]
        );
    }

    /**
     * Get products filtered by category.
     *
     * @param string $category
     * @param int    $limit
     * @return array
     */
    public function findByCategory(string $category, int $limit = 20): array
    {
        return $this->db->select(
            "SELECT * FROM products
             WHERE category = :category AND status = 'active'
             ORDER BY createdat DESC
             LIMIT :limit",
            ['category' => $category, 'limit' => $limit]
        );
    }

    /**
     * Search products by name or SKU (LIKE search).
     *
     * @param string $term  Search keyword
     * @return array
     */
    public function search(string $term): array
    {
        return $this->db->select(
            "SELECT * FROM products
             WHERE name LIKE :search OR sku LIKE :search
             ORDER BY name ASC",
            ['search' => '%' . $term . '%']
        );
    }

    /**
     * Get products with stock at or below a threshold.
     *
     * @param int $threshold  Default: 10 units
     * @return array
     */
    public function getLowStock(int $threshold = 10): array
    {
        return $this->db->select(
            "SELECT * FROM products
             WHERE stock <= :threshold AND status = 'active'
             ORDER BY stock ASC",
            ['threshold' => $threshold]
        );
    }

    /**
     * Find a product by its unique SKU.
     */
    public function findBySku(string $sku): ?array
    {
        return $this->db->first(
            "SELECT * FROM products WHERE sku = :sku",
            ['sku' => $sku]
        );
    }

    // ----------------------------------------------------------------
    // WRITE Methods
    // ----------------------------------------------------------------

    /**
     * Insert a new product and return its ID.
     *
     * @param array $data  Must contain: name, sku, price
     * @return int         The new product's auto-increment ID
     */
    public function create(array $data): int
    {
        return $this->db->insert(
            "INSERT INTO products (name, sku, price, stock, category, description, status)
             VALUES (:name, :sku, :price, :stock, :category, :description, :status)",
            [
                'name'        => $data['name'],
                'sku'         => $data['sku'],
                'price'       => $data['price'],
                'stock'       => $data['stock']       ?? 0,
                'category'    => $data['category']    ?? null,
                'description' => $data['description'] ?? null,
                'status'      => $data['status']      ?? 'active',
            ]
        );
    }

    /**
     * Partially update a product. Only the fields present in $data are changed.
     *
     * @param int   $id
     * @param array $data  Key-value pairs of columns to update
     * @return bool        True if at least one row was changed
     */
    public function update(int $id, array $data): bool
    {
        $fields = [];
        $params = ['id' => $id];

        $allowed = ['name', 'price', 'stock', 'category', 'description', 'status'];

        foreach ($allowed as $field) {
            if (array_key_exists($field, $data)) {
                $fields[]       = "{$field} = :{$field}";
                $params[$field] = $data[$field];
            }
        }

        if (empty($fields)) {
            return false; // Nothing to update
        }

        $affected = $this->db->update(
            "UPDATE products SET " . implode(', ', $fields) . " WHERE id = :id",
            $params
        );

        return $affected > 0;
    }

    /**
     * Permanently delete a product.
     *
     * @param int $id
     * @return bool  True if the row was found and deleted
     */
    public function delete(int $id): bool
    {
        return $this->db->delete(
            "DELETE FROM products WHERE id = :id",
            ['id' => $id]
        ) > 0;
    }

    // ----------------------------------------------------------------
    // Stock Management Methods
    // ----------------------------------------------------------------

    /**
     * Atomically reduce stock — only succeeds if enough stock exists.
     * Returns false (without error) if stock is insufficient.
     *
     * @param int $productId
     * @param int $quantity   Units to deduct
     * @return bool
     */
    public function reduceStock(int $productId, int $quantity): bool
    {
        $affected = $this->db->update(
            "UPDATE products
             SET stock = stock - :quantity
             WHERE id = :id AND stock >= :quantity",
            ['id' => $productId, 'quantity' => $quantity]
        );
        return $affected > 0; // false if insufficient stock
    }

    /**
     * Restore stock (e.g., on order cancellation).
     *
     * @param int $productId
     * @param int $quantity
     * @return bool
     */
    public function increaseStock(int $productId, int $quantity): bool
    {
        return $this->db->update(
            "UPDATE products SET stock = stock + :quantity WHERE id = :id",
            ['id' => $productId, 'quantity' => $quantity]
        ) > 0;
    }
}

OrderRepository — Complex Queries with JOINs

<?php
// src/api/repositories/OrderRepository.php

namespace App\Repositories;

use App\Core\Database;

/**
 * OrderRepository
 * Handles data access for orders, order_items, and order_transactions tables.
 */
class OrderRepository
{
    private Database $db;

    public function __construct(Database $db)
    {
        $this->db = $db;
    }

    // ----------------------------------------------------------------
    // READ Methods
    // ----------------------------------------------------------------

    /**
     * Find an order by ID.
     */
    public function findById(int $id): ?array
    {
        return $this->db->first(
            "SELECT * FROM orders WHERE id = :id",
            ['id' => $id]
        );
    }

    /**
     * Get all orders for a specific user, newest first.
     *
     * @param int $userId
     * @param int $limit
     * @return array
     */
    public function findByUser(int $userId, int $limit = 20): array
    {
        return $this->db->select(
            "SELECT * FROM orders
             WHERE userid = :userid
             ORDER BY createdat DESC
             LIMIT :limit",
            ['userid' => $userId, 'limit' => $limit]
        );
    }

    /**
     * Get order items joined with product details — a JOIN query.
     *
     * @param int $orderId
     * @return array  Each row has order item fields + product name and SKU
     */
    public function getOrderItems(int $orderId): array
    {
        return $this->db->select(
            "SELECT
                oi.id,
                oi.orderid,
                oi.productid,
                oi.quantity,
                oi.price,
                oi.subtotal,
                p.name  AS productname,
                p.sku   AS productsku,
                p.stock AS currentstock
             FROM order_items oi
             INNER JOIN products p ON oi.productid = p.id
             WHERE oi.orderid = :orderid
             ORDER BY oi.id ASC",
            ['orderid' => $orderId]
        );
    }

    /**
     * Get a full order with user info (JOIN with users table).
     */
    public function findWithUser(int $orderId): ?array
    {
        return $this->db->first(
            "SELECT
                o.*,
                u.name  AS username,
                u.email AS useremail,
                u.phone AS userphone
             FROM orders o
             INNER JOIN users u ON o.userid = u.id
             WHERE o.id = :id",
            ['id' => $orderId]
        );
    }

    /**
     * Get orders by status (e.g., all 'pending' orders).
     */
    public function findByStatus(string $status): array
    {
        return $this->db->select(
            "SELECT o.*, u.name AS username, u.email AS useremail
             FROM orders o
             INNER JOIN users u ON o.userid = u.id
             WHERE o.status = :status
             ORDER BY o.createdat ASC",
            ['status' => $status]
        );
    }

    /**
     * Get transaction log entries for an order.
     */
    public function getTransactionLog(int $orderId): array
    {
        return $this->db->select(
            "SELECT * FROM order_transactions
             WHERE orderid = :orderid
             ORDER BY createdat DESC",
            ['orderid' => $orderId]
        );
    }

    /**
     * Get aggregate order statistics — uses COUNT, SUM, AVG, and CASE WHEN.
     *
     * @return array  Single row with stats
     */
    public function getStatistics(): array
    {
        $result = $this->db->first(
            "SELECT
                COUNT(*)                                                  AS total_orders,
                SUM(CASE WHEN status = 'pending'   THEN 1 ELSE 0 END)     AS pending_orders,
                SUM(CASE WHEN status = 'completed' THEN 1 ELSE 0 END)     AS completed_orders,
                SUM(CASE WHEN status = 'cancelled' THEN 1 ELSE 0 END)     AS cancelled_orders,
                COALESCE(SUM(totalamount), 0)                             AS total_revenue,
                COALESCE(AVG(totalamount), 0)                             AS average_order_value
             FROM orders"
        );
        return $result ?? [];
    }

    // ----------------------------------------------------------------
    // WRITE Methods
    // ----------------------------------------------------------------

    /**
     * Create a new order and return its ID.
     *
     * @param array $data
     * @return int  New order ID
     */
    public function create(array $data): int
    {
        return $this->db->insert(
            "INSERT INTO orders (userid, totalamount, paymentmethod, status, notes)
             VALUES (:userid, :totalamount, :paymentmethod, :status, :notes)",
            [
                'userid'        => $data['userid'],
                'totalamount'   => $data['totalamount'],
                'paymentmethod' => $data['paymentmethod'],
                'status'        => $data['status']  ?? 'pending',
                'notes'         => $data['notes']   ?? null,
            ]
        );
    }

    /**
     * Add a single item line to an order.
     *
     * @param array $data  Must contain: orderid, productid, quantity, price, subtotal
     * @return int  New order_item ID
     */
    public function addOrderItem(array $data): int
    {
        return $this->db->insert(
            "INSERT INTO order_items (orderid, productid, quantity, price, subtotal)
             VALUES (:orderid, :productid, :quantity, :price, :subtotal)",
            [
                'orderid'   => $data['orderid'],
                'productid' => $data['productid'],
                'quantity'  => $data['quantity'],
                'price'     => $data['price'],
                'subtotal'  => $data['subtotal'],
            ]
        );
    }

    /**
     * Update order fields (partial update pattern).
     *
     * @param int   $id
     * @param array $data
     * @return bool
     */
    public function update(int $id, array $data): bool
    {
        $fields = [];
        $params = ['id' => $id];

        $allowed = ['status', 'cancellationreason', 'cancelledat', 'notes'];

        foreach ($allowed as $field) {
            if (array_key_exists($field, $data)) {
                $fields[]       = "{$field} = :{$field}";
                $params[$field] = $data[$field];
            }
        }

        if (empty($fields)) return false;

        return $this->db->update(
            "UPDATE orders SET " . implode(', ', $fields) . " WHERE id = :id",
            $params
        ) > 0;
    }

    /**
     * Append a transaction log entry for an order.
     *
     * @param array $data
     * @return int  New transaction ID
     */
    public function createTransactionLog(array $data): int
    {
        return $this->db->insert(
            "INSERT INTO order_transactions
                (orderid, action, amount, paymentmethod, status, notes)
             VALUES
                (:orderid, :action, :amount, :paymentmethod, :status, :notes)",
            [
                'orderid'       => $data['orderid'],
                'action'        => $data['action'],
                'amount'        => $data['amount'],
                'paymentmethod' => $data['paymentmethod'],
                'status'        => $data['status'],
                'notes'         => $data['notes'] ?? null,
            ]
        );
    }
}

Advanced Query Patterns

JOIN Queries

// INNER JOIN — get only rows that match in both tables
public function getOrdersWithUsers(): array
{
    return $this->db->select(
        "SELECT
            o.id,
            o.totalamount,
            o.status,
            o.createdat,
            u.name  AS username,
            u.email AS useremail
         FROM orders o
         INNER JOIN users u ON o.userid = u.id
         ORDER BY o.createdat DESC"
    );
}

// LEFT JOIN — include products even if they have no order items
public function getProductsWithOrderCount(): array
{
    return $this->db->select(
        "SELECT
            p.id,
            p.name,
            p.sku,
            COUNT(oi.id)      AS times_ordered,
            COALESCE(SUM(oi.quantity), 0) AS total_units_sold
         FROM products p
         LEFT JOIN order_items oi ON p.id = oi.productid
         GROUP BY p.id, p.name, p.sku
         ORDER BY total_units_sold DESC"
    );
}

Aggregate Queries

// Revenue and sales summary for a single product
public function getProductSalesSummary(int $productId): ?array
{
    return $this->db->first(
        "SELECT
            p.id,
            p.name,
            p.price                        AS current_price,
            COUNT(oi.id)                   AS times_ordered,
            COALESCE(SUM(oi.quantity), 0)  AS total_units_sold,
            COALESCE(SUM(oi.subtotal), 0)  AS total_revenue
         FROM products p
         LEFT JOIN order_items oi ON p.id = oi.productid
         WHERE p.id = :productid
         GROUP BY p.id",
        ['productid' => $productId]
    );
}

// Daily revenue for the last 30 days
public function getDailyRevenue(): array
{
    return $this->db->select(
        "SELECT
            DATE(createdat)    AS order_date,
            COUNT(*)           AS order_count,
            SUM(totalamount)   AS daily_revenue
         FROM orders
         WHERE status = 'completed'
           AND createdat >= DATE_SUB(NOW(), INTERVAL 30 DAY)
         GROUP BY DATE(createdat)
         ORDER BY order_date DESC"
    );
}

Subqueries

// Get products that have NEVER been ordered
public function getProductsNeverOrdered(): array
{
    return $this->db->select(
        "SELECT * FROM products
         WHERE id NOT IN (
             SELECT DISTINCT productid FROM order_items
         )
         ORDER BY name ASC"
    );
}

// Get users who placed more than 5 orders
public function getFrequentBuyers(int $minOrders = 5): array
{
    return $this->db->select(
        "SELECT u.id, u.name, u.email, COUNT(o.id) AS order_count
         FROM users u
         INNER JOIN orders o ON u.id = o.userid
         GROUP BY u.id, u.name, u.email
         HAVING order_count >= :min
         ORDER BY order_count DESC",
        ['min' => $minOrders]
    );
}

Conditional / Atomic Updates

// Atomically deduct stock only if enough exists
// The WHERE clause makes this safe even under concurrent requests
public function deductStock(int $productId, int $qty): bool
{
    return $this->db->update(
        "UPDATE products
         SET stock = stock - :qty
         WHERE id = :id AND stock >= :qty",
        ['id' => $productId, 'qty' => $qty]
    ) > 0;
    // Returns false if product not found OR stock insufficient
    // No separate SELECT needed — the condition is in the WHERE clause
}

// Soft-delete (mark as deleted instead of removing the row)
public function softDelete(int $id): bool
{
    return $this->db->update(
        "UPDATE products SET status = 'deleted', deletedat = NOW() WHERE id = :id",
        ['id' => $id]
    ) > 0;
}

Dynamic WHERE Clause (Partial Update Pattern)

// Build an UPDATE statement from only the fields that were provided
// This avoids overwriting fields the caller didn't intend to change
public function update(int $id, array $data): bool
{
    $fields = [];
    $params = ['id' => $id];

    // Whitelist which columns are updatable
    $allowed = ['name', 'price', 'stock', 'category', 'status', 'description'];

    foreach ($allowed as $column) {
        if (array_key_exists($column, $data)) {
            $fields[]        = "{$column} = :{$column}";
            $params[$column] = $data[$column];
        }
    }

    if (empty($fields)) {
        return false; // No updatable fields were provided
    }

    // Example output: "UPDATE products SET price = :price, stock = :stock WHERE id = :id"
    $affected = $this->db->update(
        "UPDATE products SET " . implode(', ', $fields) . " WHERE id = :id",
        $params
    );

    return $affected > 0;
}

Repository Best Practices

✓ DO

Always use :named parameter binding — never concatenate user input
Use first() for single-row lookups — it is more expressive and adds LIMIT 1 automatically
Return null from findById() when nothing is found
Return [] (empty array) from list methods when no results exist
Use descriptive method names: findByEmail(), getLowStock(), findByStatus()
Whitelist allowed columns in partial-update methods to prevent mass-assignment
Use atomic SQL conditions (e.g., WHERE stock >= :qty) for race condition safety
Add PHPDoc comments on complex queries explaining what data is returned

✗ DON'T

Concatenate user input into SQL — "WHERE id = $id" is always wrong
Return PDOStatement objects — always return arrays
Perform business logic, price calculations, or validation
Call other repositories (services orchestrate repositories)
Manage transactions — services call beginTransaction(), not repositories
Throw ValidationException — repositories only throw DatabaseException
Write overly complex queries that do the job of two simpler ones

Common Patterns Summary

Find Single Record

public function findById(int $id): ?array
{
    return $this->db->first(
        "SELECT * FROM table_name WHERE id = :id",
        ['id' => $id]
    );
    // Returns the row array, or null if not found
}

Find Multiple Records

public function findAll(): array
{
    return $this->db->select(
        "SELECT * FROM table_name ORDER BY createdat DESC"
    );
    // Returns array of rows; [] if table is empty
}

Create a Record

public function create(array $data): int
{
    return $this->db->insert(
        "INSERT INTO table_name (field1, field2) VALUES (:field1, :field2)",
        [
            'field1' => $data['field1'],
            'field2' => $data['field2'],
        ]
    );
    // Returns the new row's auto-increment ID
}

Update a Record

public function update(int $id, array $data): bool
{
    $affected = $this->db->update(
        "UPDATE table_name SET field1 = :field1, field2 = :field2 WHERE id = :id",
        ['id' => $id, 'field1' => $data['field1'], 'field2' => $data['field2']]
    );
    return $affected > 0;
    // true if updated, false if not found or values unchanged
}

Delete a Record

public function delete(int $id): bool
{
    return $this->db->delete(
        "DELETE FROM table_name WHERE id = :id",
        ['id' => $id]
    ) > 0;
    // true if deleted, false if not found
}
