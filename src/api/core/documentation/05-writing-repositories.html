Writing Repositories

Learn how to implement the data access layer with repositories

What is a Repository?

A Repository is a class that handles all database operations for a specific entity or domain. Repositories are the only place in your application where SQL queries should be written.

Three-Layer Architecture

Controller Layer
  • Handles HTTP requests
  • Calls Service methods
  • Returns formatted responses
Service Layer
  • Contains business logic
  • Orchestrates repositories
  • Manages transactions
Repository Layer (YOU ARE HERE)
  • Executes SQL queries
  • Maps database results to arrays
  • Provides data access methods
  • No business logic

Why Repositories Exist

Benefits of Repository Pattern

  • Single Source of Truth: All SQL for an entity is in one place
  • Testability: Easy to mock repositories in tests
  • Maintainability: Changing a query affects only one file
  • Security: Centralized parameter binding prevents SQL injection
  • Reusability: Same query methods used across services

Repository Layer Responsibilities

✓ Repositories SHOULD

Write SQL queries (SELECT, INSERT, UPDATE, DELETE)
Use parameter binding for all user input
Return arrays or null (not PDO objects)
Provide CRUD methods (Create, Read, Update, Delete)
Map database results to meaningful structures
Handle database-specific operations

✗ Repositories SHOULD NOT

Contain business logic
Validate business rules
Handle transactions (services do this)
Call other repositories
Throw ValidationException (only DatabaseException)
Calculate derived values (services do this)

Creating Your First Repository

File Location and Naming

  • Location: src/api/repositories/
  • Naming: *Repository.php (e.g., ProductRepository.php)
  • Class Name: Must match filename (e.g., class ProductRepository)

Basic Repository Template

<?php
// src/api/repositories/ProductRepository.php

namespace App\Repositories;

use App\Core\Database;
use App\Core\Exceptions\DatabaseException;

class ProductRepository
{
    private Database $db;
    
    /**
     * Constructor - Accept Database instance
     */
    public function __construct(Database $db)
    {
        $this->db = $db;
    }
    
    /**
     * Find product by ID
     */
    public function findById(int $id): ?array
    {
        $sql = "SELECT * FROM products WHERE id = :id";
        $result = $this->db->query($sql, ['id' => $id]);
        return $result[0] ?? null;
    }
    
    /**
     * Get all products
     */
    public function findAll(): array
    {
        $sql = "SELECT * FROM products ORDER BY createdat DESC";
        return $this->db->query($sql);
    }
    
    /**
     * Create new product
     */
    public function create(array $data): int
    {
        $sql = "INSERT INTO products (name, sku, price, stock, category) 
                VALUES (:name, :sku, :price, :stock, :category)";
        
        return $this->db->insert($sql, [
            'name' => $data['name'],
            'sku' => $data['sku'],
            'price' => $data['price'],
            'stock' => $data['stock'] ?? 0,
            'category' => $data['category'] ?? null
        ]);
    }
    
    /**
     * Update product
     */
    public function update(int $id, array $data): bool
    {
        $sql = "UPDATE products 
                SET name = :name, price = :price, stock = :stock
                WHERE id = :id";
        
        $affected = $this->db->update($sql, [
            'id' => $id,
            'name' => $data['name'],
            'price' => $data['price'],
            'stock' => $data['stock']
        ]);
        
        return $affected > 0;
    }
    
    /**
     * Delete product
     */
    public function delete(int $id): bool
    {
        $sql = "DELETE FROM products WHERE id = :id";
        $affected = $this->db->delete($sql, ['id' => $id]);
        return $affected > 0;
    }
}

Using the Database Class

The framework provides a Database class with methods for all database operations. All repositories receive a Database instance in their constructor.

query() - Execute SELECT Queries

public function query(string $sql, array $params = []): array

Use for: SELECT queries that return multiple rows

Returns: Array of associative arrays (or empty array)

// Get all products
$sql = "SELECT * FROM products";
$products = $this->db->query($sql);
// Returns: [['id' => 1, 'name' => 'Laptop'], ['id' => 2, 'name' => 'Phone']]

// Get products with filter
$sql = "SELECT * FROM products WHERE category = :category";
$products = $this->db->query($sql, ['category' => 'Electronics']);

// Get single product (returns array with one element)
$sql = "SELECT * FROM products WHERE id = :id";
$result = $this->db->query($sql, ['id' => 123]);
$product = $result[0] ?? null;  // Get first element or null

insert() - Insert and Return ID

public function insert(string $sql, array $params = []): int

Use for: INSERT queries where you need the new record's ID

Returns: The auto-increment ID of the inserted row

$sql = "INSERT INTO products (name, sku, price) 
        VALUES (:name, :sku, :price)";

$productId = $this->db->insert($sql, [
    'name' => 'Laptop',
    'sku' => 'LAP-001',
    'price' => 50000
]);

// $productId = 123 (the new product's ID)

update() - Update Records

public function update(string $sql, array $params = []): int

Use for: UPDATE queries

Returns: Number of affected rows

$sql = "UPDATE products SET price = :price WHERE id = :id";

$affected = $this->db->update($sql, [
    'id' => 123,
    'price' => 45000
]);

// $affected = 1 (if product exists and was updated)
// $affected = 0 (if product doesn't exist or price was already 45000)

delete() - Delete Records

public function delete(string $sql, array $params = []): int

Use for: DELETE queries

Returns: Number of affected rows

$sql = "DELETE FROM products WHERE id = :id";

$affected = $this->db->delete($sql, ['id' => 123]);

// $affected = 1 (if product existed and was deleted)
// $affected = 0 (if product didn't exist)

execute() - General Purpose Execution

public function execute(string $sql, array $params = []): bool

Use for: Any query where you just need success/failure (not commonly used)

Returns: Boolean (true on success, false on failure)

$sql = "UPDATE products SET stock = stock + :quantity WHERE id = :id";

$success = $this->db->execute($sql, [
    'id' => 123,
    'quantity' => 10
]);

// $success = true or false

Parameter Binding (SQL Injection Prevention)

⚠️ CRITICAL: Always Use Parameter Binding

Parameter binding prevents SQL injection attacks. Never concatenate user input into SQL strings.

Named Parameters

// ✓ GOOD: Using named parameters
$sql = "SELECT * FROM products WHERE category = :category AND price > :minPrice";
$products = $this->db->query($sql, [
    'category' => $userInput,
    'minPrice' => 1000
]);

// ❌ BAD: String concatenation (VULNERABLE TO SQL INJECTION!)
$sql = "SELECT * FROM products WHERE category = '$userInput'";
$products = $this->db->query($sql);  // NEVER DO THIS!

How Parameter Binding Works

  1. Write SQL with :placeholder syntax
  2. Pass parameters as associative array
  3. PDO safely escapes and binds values
  4. SQL injection is impossible
// Step 1: SQL with placeholders
$sql = "INSERT INTO users (name, email, age) VALUES (:name, :email, :age)";

// Step 2: Parameters array
$params = [
    'name' => $userData['name'],      // Automatically escaped
    'email' => $userData['email'],    // Automatically escaped
    'age' => $userData['age']         // Automatically escaped
];

// Step 3: Execute safely
$userId = $this->db->insert($sql, $params);

Complete Repository Examples

ProductRepository - Full Implementation

<?php

namespace App\Repositories;

use App\Core\Database;

/**
 * ProductRepository - Data access for products table
 */
class ProductRepository
{
    private Database $db;
    
    public function __construct(Database $db)
    {
        $this->db = $db;
    }
    
    /**
     * Find product by ID
     * 
     * @param int $id Product ID
     * @return array|null Product data or null
     */
    public function findById(int $id): ?array
    {
        $sql = "SELECT * FROM products WHERE id = :id";
        $result = $this->db->query($sql, ['id' => $id]);
        return $result[0] ?? null;
    }
    
    /**
     * Get all products with optional filters
     * 
     * @param string|null $category Optional category filter
     * @param int $limit Number of products to return
     * @return array Products list
     */
    public function getAll(?string $category = null, int $limit = 10): array
    {
        if ($category) {
            $sql = "SELECT * FROM products 
                    WHERE category = :category 
                    ORDER BY createdat DESC 
                    LIMIT :limit";
            
            return $this->db->query($sql, [
                'category' => $category,
                'limit' => $limit
            ]);
        }
        
        $sql = "SELECT * FROM products 
                ORDER BY createdat DESC 
                LIMIT :limit";
        
        return $this->db->query($sql, ['limit' => $limit]);
    }
    
    /**
     * Create new product
     * 
     * @param array $data Product data
     * @return int New product ID
     */
    public function create(array $data): int
    {
        $sql = "INSERT INTO products (name, sku, price, stock, category, description, status) 
                VALUES (:name, :sku, :price, :stock, :category, :description, :status)";
        
        return $this->db->insert($sql, [
            'name' => $data['name'],
            'sku' => $data['sku'],
            'price' => $data['price'],
            'stock' => $data['stock'] ?? 0,
            'category' => $data['category'] ?? null,
            'description' => $data['description'] ?? null,
            'status' => $data['status'] ?? 'active'
        ]);
    }
    
    /**
     * Update product
     * 
     * @param int $id Product ID
     * @param array $data Data to update
     * @return bool Success status
     */
    public function update(int $id, array $data): bool
    {
        // Build dynamic update query based on provided fields
        $fields = [];
        $params = ['id' => $id];
        
        if (isset($data['name'])) {
            $fields[] = "name = :name";
            $params['name'] = $data['name'];
        }
        
        if (isset($data['price'])) {
            $fields[] = "price = :price";
            $params['price'] = $data['price'];
        }
        
        if (isset($data['stock'])) {
            $fields[] = "stock = :stock";
            $params['stock'] = $data['stock'];
        }
        
        if (isset($data['category'])) {
            $fields[] = "category = :category";
            $params['category'] = $data['category'];
        }
        
        if (isset($data['status'])) {
            $fields[] = "status = :status";
            $params['status'] = $data['status'];
        }
        
        if (empty($fields)) {
            return false;  // Nothing to update
        }
        
        $sql = "UPDATE products SET " . implode(', ', $fields) . " WHERE id = :id";
        $affected = $this->db->update($sql, $params);
        
        return $affected > 0;
    }
    
    /**
     * Delete product
     * 
     * @param int $id Product ID
     * @return bool Success status
     */
    public function delete(int $id): bool
    {
        $sql = "DELETE FROM products WHERE id = :id";
        $affected = $this->db->delete($sql, ['id' => $id]);
        return $affected > 0;
    }
    
    /**
     * Reduce product stock (for orders)
     * 
     * @param int $productId Product ID
     * @param int $quantity Quantity to reduce
     * @return bool Success status
     */
    public function reduceStock(int $productId, int $quantity): bool
    {
        $sql = "UPDATE products 
                SET stock = stock - :quantity 
                WHERE id = :id AND stock >= :quantity";
        
        $affected = $this->db->update($sql, [
            'id' => $productId,
            'quantity' => $quantity
        ]);
        
        return $affected > 0;
    }
    
    /**
     * Increase product stock (for cancellations)
     * 
     * @param int $productId Product ID
     * @param int $quantity Quantity to increase
     * @return bool Success status
     */
    public function increaseStock(int $productId, int $quantity): bool
    {
        $sql = "UPDATE products 
                SET stock = stock + :quantity 
                WHERE id = :id";
        
        $affected = $this->db->update($sql, [
            'id' => $productId,
            'quantity' => $quantity
        ]);
        
        return $affected > 0;
    }
    
    /**
     * Search products by name or SKU
     * 
     * @param string $searchTerm Search term
     * @return array Matching products
     */
    public function search(string $searchTerm): array
    {
        $sql = "SELECT * FROM products 
                WHERE name LIKE :search OR sku LIKE :search
                ORDER BY name ASC";
        
        return $this->db->query($sql, [
            'search' => '%' . $searchTerm . '%'
        ]);
    }
    
    /**
     * Get products with low stock
     * 
     * @param int $threshold Stock threshold
     * @return array Products with low stock
     */
    public function getLowStockProducts(int $threshold = 10): array
    {
        $sql = "SELECT * FROM products 
                WHERE stock <= :threshold AND status = 'active'
                ORDER BY stock ASC";
        
        return $this->db->query($sql, ['threshold' => $threshold]);
    }
}

OrderRepository - Complex Queries with Joins

<?php

namespace App\Repositories;

use App\Core\Database;

/**
 * OrderRepository - Data access for orders, order_items, and transactions
 */
class OrderRepository
{
    private Database $db;
    
    public function __construct(Database $db)
    {
        $this->db = $db;
    }
    
    /**
     * Find order by ID
     * 
     * @param int $id Order ID
     * @return array|null Order data or null
     */
    public function findById(int $id): ?array
    {
        $sql = "SELECT * FROM orders WHERE id = :id";
        $result = $this->db->query($sql, ['id' => $id]);
        return $result[0] ?? null;
    }
    
    /**
     * Create new order
     * 
     * @param array $data Order data
     * @return int New order ID
     */
    public function create(array $data): int
    {
        $sql = "INSERT INTO orders (userid, totalamount, paymentmethod, status, notes) 
                VALUES (:userid, :totalamount, :paymentmethod, :status, :notes)";
        
        return $this->db->insert($sql, [
            'userid' => $data['userid'],
            'totalamount' => $data['totalamount'],
            'paymentmethod' => $data['paymentmethod'],
            'status' => $data['status'] ?? 'pending',
            'notes' => $data['notes'] ?? null
        ]);
    }
    
    /**
     * Update order
     * 
     * @param int $id Order ID
     * @param array $data Data to update
     * @return bool Success status
     */
    public function update(int $id, array $data): bool
    {
        $fields = [];
        $params = ['id' => $id];
        
        if (isset($data['status'])) {
            $fields[] = "status = :status";
            $params['status'] = $data['status'];
        }
        
        if (isset($data['cancellationreason'])) {
            $fields[] = "cancellationreason = :cancellationreason";
            $params['cancellationreason'] = $data['cancellationreason'];
        }
        
        if (isset($data['cancelledat'])) {
            $fields[] = "cancelledat = :cancelledat";
            $params['cancelledat'] = $data['cancelledat'];
        }
        
        if (empty($fields)) {
            return false;
        }
        
        $sql = "UPDATE orders SET " . implode(', ', $fields) . " WHERE id = :id";
        $affected = $this->db->update($sql, $params);
        
        return $affected > 0;
    }
    
    /**
     * Add item to order
     * 
     * @param array $data Order item data
     * @return int New item ID
     */
    public function addOrderItem(array $data): int
    {
        $sql = "INSERT INTO order_items (orderid, productid, quantity, price, subtotal) 
                VALUES (:orderid, :productid, :quantity, :price, :subtotal)";
        
        return $this->db->insert($sql, [
            'orderid' => $data['orderid'],
            'productid' => $data['productid'],
            'quantity' => $data['quantity'],
            'price' => $data['price'],
            'subtotal' => $data['subtotal']
        ]);
    }
    
    /**
     * Get order items with product details (JOIN query)
     * 
     * @param int $orderId Order ID
     * @return array Order items with product info
     */
    public function getOrderItems(int $orderId): array
    {
        $sql = "SELECT 
                    oi.id,
                    oi.orderid,
                    oi.productid,
                    oi.quantity,
                    oi.price,
                    oi.subtotal,
                    p.name AS productname,
                    p.sku AS productsku
                FROM order_items oi
                INNER JOIN products p ON oi.productid = p.id
                WHERE oi.orderid = :orderid
                ORDER BY oi.id ASC";
        
        return $this->db->query($sql, ['orderid' => $orderId]);
    }
    
    /**
     * Get orders by user
     * 
     * @param int $userId User ID
     * @param int $limit Number of orders to return
     * @return array User's orders
     */
    public function getByUser(int $userId, int $limit = 10): array
    {
        $sql = "SELECT * FROM orders 
                WHERE userid = :userid 
                ORDER BY createdat DESC 
                LIMIT :limit";
        
        return $this->db->query($sql, [
            'userid' => $userId,
            'limit' => $limit
        ]);
    }
    
    /**
     * Create transaction log entry
     * 
     * @param array $data Transaction data
     * @return int New transaction ID
     */
    public function createTransactionLog(array $data): int
    {
        $sql = "INSERT INTO order_transactions 
                (orderid, action, amount, paymentmethod, status, notes) 
                VALUES (:orderid, :action, :amount, :paymentmethod, :status, :notes)";
        
        return $this->db->insert($sql, [
            'orderid' => $data['orderid'],
            'action' => $data['action'],
            'amount' => $data['amount'],
            'paymentmethod' => $data['paymentmethod'],
            'status' => $data['status'],
            'notes' => $data['notes'] ?? null
        ]);
    }
    
    /**
     * Get transaction log for order
     * 
     * @param int $orderId Order ID
     * @return array Transaction log entries
     */
    public function getTransactionLog(int $orderId): array
    {
        $sql = "SELECT * FROM order_transactions 
                WHERE orderid = :orderid 
                ORDER BY createdat DESC";
        
        return $this->db->query($sql, ['orderid' => $orderId]);
    }
    
    /**
     * Get order statistics (advanced query)
     * 
     * @return array Order statistics
     */
    public function getOrderStatistics(): array
    {
        $sql = "SELECT 
                    COUNT(*) AS total_orders,
                    SUM(CASE WHEN status = 'pending' THEN 1 ELSE 0 END) AS pending_orders,
                    SUM(CASE WHEN status = 'completed' THEN 1 ELSE 0 END) AS completed_orders,
                    SUM(CASE WHEN status = 'cancelled' THEN 1 ELSE 0 END) AS cancelled_orders,
                    SUM(totalamount) AS total_revenue,
                    AVG(totalamount) AS average_order_value
                FROM orders";
        
        $result = $this->db->query($sql);
        return $result[0] ?? [];
    }
}

Advanced Query Patterns

JOIN Queries

// Get orders with user information
public function getOrdersWithUsers(): array
{
    $sql = "SELECT 
                o.id,
                o.totalamount,
                o.status,
                o.createdat,
                u.name AS username,
                u.email AS useremail
            FROM orders o
            INNER JOIN users u ON o.userid = u.id
            ORDER BY o.createdat DESC";
    
    return $this->db->query($sql);
}

Aggregate Queries

// Get product sales summary
public function getProductSalesSummary(int $productId): ?array
{
    $sql = "SELECT 
                p.id,
                p.name,
                COUNT(oi.id) AS times_ordered,
                SUM(oi.quantity) AS total_quantity_sold,
                SUM(oi.subtotal) AS total_revenue
            FROM products p
            LEFT JOIN order_items oi ON p.id = oi.productid
            WHERE p.id = :productid
            GROUP BY p.id";
    
    $result = $this->db->query($sql, ['productid' => $productId]);
    return $result[0] ?? null;
}

Subqueries

// Get products never ordered
public function getProductsNeverOrdered(): array
{
    $sql = "SELECT * FROM products 
            WHERE id NOT IN (
                SELECT DISTINCT productid FROM order_items
            )
            ORDER BY name ASC";
    
    return $this->db->query($sql);
}

Conditional Updates

// Update stock only if sufficient quantity available
public function reduceStockIfAvailable(int $productId, int $quantity): bool
{
    $sql = "UPDATE products 
            SET stock = stock - :quantity 
            WHERE id = :id 
            AND stock >= :quantity";  // Condition in WHERE
    
    $affected = $this->db->update($sql, [
        'id' => $productId,
        'quantity' => $quantity
    ]);
    
    return $affected > 0;  // Returns false if insufficient stock
}

Repository Best Practices

✓ DO

Always use parameter binding for user input
Return null for findById when not found
Return empty arrays [] for list queries with no results
Use meaningful method names (findById, getAll)
Document complex queries with comments
Keep queries focused and simple
Use SELECT * for simplicity (optimize later if needed)

✗ DON'T

Concatenate user input into SQL strings
Return PDOStatement objects (always return arrays)
Perform business logic calculations
Call other repositories
Handle transactions (services do this)
Throw ValidationException (only DatabaseException if needed)
Write overly complex queries that should be split

Return Value Conventions

Method Type Return Type When Not Found Example
findById() ?array null ['id' => 1, 'name' => 'Laptop']
findAll() / getAll() array [] (empty array) [['id' => 1], ['id' => 2]]
create() int N/A (throws exception) 123 (new ID)
update() bool false true or false
delete() bool false true or false
